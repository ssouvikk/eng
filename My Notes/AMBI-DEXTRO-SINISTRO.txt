DEXTRO-    → right hand, right side
SINISTRO-  → left hand, left side
DROIT      → right, skill, expertise

ambiguous      → having more than one possible meaning
ambidextrous   → able to use both hands equally well
ambidexterity  → the ability to use both hands equally well
ambivalent     → having mixed or conflicting feelings about something
ambition       → a strong desire to achieve success or a goal
ambivert       → a person who has traits of both an introvert and an extrovert

dextrous       → skillful; showing manual or mental skill
dexterity      → skill, especially with the hands
adroit         → skillful, clever, expert
adroitness     → skillfulness; expertise
droit-handed   → right-handed; skilled in using the right hand

sinister       → threatening, evil, ominous; originally "on the left side"
gauche         → socially awkward; lacking grace or sophistication

Memory Trick:

LEFT SIDE  → sinister, gauche → awkward, clumsy, threatening
RIGHT SIDE → dextrous, adroit, droit → skillful, expert
BOTH SIDES → ambidextrous → able to use both hands
