AMBI-         → both, on both sides
DEXTRO-       → right hand, right side
SINISTRO-     → left hand, left side
DROIT         → right, skill, expertise
GAUCHE        → left; awkward, lacking grace

-GUOUS        → leading, driving
-VALENT       → strength, force
-VERT         → turn
-ITY          → state, quality
-NESS         → state, condition
======================================================

AMBIGUOUS       → having more than one possible meaning
AMBIDEXTROUS    → able to use both hands equally well
AMBIDEXTERITY   → ability to use both hands equally well
AMBIVALENT      → having conflicting feelings
AMBITION        → strong desire for achievement
AMBIVERT        → person with both introverted and extroverted traits

DEXTROUS        → skillful; adept
DEXTERITY       → manual skill and agility
ADROIT          → skillful; clever
ADROITNESS      → expertise; skillfulness
DROIT-HANDED    → right-handed

SINISTER        → threatening; ominous; evil
GAUCHE          → socially awkward; lacking sophistication

======================================================

AMBIGUOUS : {
    ambi (both) + guous (leading)

    Semantic Evolution (literal → modern):
        → leading in two directions
        → open to multiple interpretations
        → having more than one possible meaning
}

AMBIDEXTROUS : {
    ambi (both) + dextrous (right-handed, skillful)

    Semantic Evolution (literal → modern):
        → right-handed on both sides
        → equally skillful with either hand
        → able to use both hands equally well
}

AMBIDEXTERITY : {
    ambi (both) + dexterity (skillfulness)

    Semantic Evolution (literal → modern):
        → skillfulness on both sides
        → equal manual ability in each hand
        → ability to use both hands equally well
}

AMBIVALENT : {
    ambi (both) + valent (strength, force)

    Semantic Evolution (literal → modern):
        → pulled by two opposing forces
        → experiencing contradictory emotions
        → having mixed or conflicting feelings
}

AMBITION : {
    Latin ambitio (going around seeking favor)

    Semantic Evolution (literal → modern):
        → seeking support from all sides
        → desire for advancement and recognition
        → strong aspiration toward success
}

AMBIVERT : {
    ambi (both) + vert (turn)

    Semantic Evolution (literal → modern):
        → turning in both directions
        → displaying introverted and extroverted tendencies
        → person with a balanced social orientation
}

DEXTROUS : {
    dextr (right hand)

    Semantic Evolution (literal → modern):
        → skillful with the right hand
        → manually adept
        → showing physical or mental skill
}

DEXTERITY : {
    dexter (right hand) + ity (quality)

    Semantic Evolution (literal → modern):
        → quality of right-handed skill
        → manual agility
        → expert coordination and skillfulness
}

ADROIT : {
    French à droit (to the right)

    Semantic Evolution (literal → modern):
        → on the right side
        → skillful and capable
        → clever and expert in action
}

ADROITNESS : {
    adroit + ness (state, quality)

    Semantic Evolution (literal → modern):
        → state of being skillful
        → expertise and competence
        → clever effectiveness in performance
}

DROIT-HANDED : {
    droit (right) + handed

    Semantic Evolution (literal → modern):
        → using the right hand preferentially
        → right-handed
        → skilled in use of the right hand
}

SINISTER : {
    sinistr (left side)

    Semantic Evolution (literal → modern):
        → located on the left side
        → associated with bad omens
        → threatening, evil, or ominous
}

GAUCHE : {
    French gauche (left)

    Semantic Evolution (literal → modern):
        → left-sided
        → clumsy or socially uncomfortable
        → lacking grace, tact, or sophistication
}

======================================================

Memory Trick:

LEFT SIDE  → sinister, gauche → awkward, clumsy, threatening
RIGHT SIDE → dextrous, adroit, droit → skillful, expert
BOTH SIDES → ambidextrous → able to use both hands
